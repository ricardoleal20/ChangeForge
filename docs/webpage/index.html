What is ChangeForge?

ChangeForge streamlines your release process by automating changelog generation based on your commit history or structured change files. It enforces a consistent format, making it easy for your users and your team to see what's new in every release. Say goodbye to manual changelog updates and hello to automated precision.

Quick Start

Get up and running with ChangeForge in just a couple of commands.

Installation & First Use Copy 
$ pip install changeforge
$ changeforge init
$ changeforge create
$ changeforge list
$ changeforge bump
